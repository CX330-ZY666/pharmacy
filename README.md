# 💊 药品存销信息管理系统

数据库课程设计项目。

## 技术栈

- 数据库：MySQL 8.x
- 后端：Python Flask + PyMySQL
- 前端：HTML + Bootstrap 5

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/你的用户名/pharmacy.git
cd pharmacy

# 2. 安装依赖
pip install flask pymysql

# 3. 配置数据库
copy config.example.py config.py
# 编辑 config.py，修改 password 为你的 MySQL 密码

# 4. 初始化数据库
# 打开 MySQL Workbench，执行 init_db.sql

# 5. 启动项目
python app.py
# 浏览器打开 http://127.0.0.1:5000
```
